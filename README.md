# Projet EasyRedmine

## Sommaire

Ce projet a pour but de créer une interface utilisateur pour interagir avec l'API d'EasyRedmine.  
L'objectif est de faciliter la gestion des projets, des tâches et des utilisateurs au sein de l'application EasyRedmine.
